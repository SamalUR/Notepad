Simple Notepad
## Features
- Text editing area.
- File menu: Open, Save, Exit.
- Edit menu: Cut, Copy, Paste.
- Help menu: About (with name and ID).

## How to Run
1. Ensure Java JDK 8 or later is installed.
2. Compile: `javac Notepad.java`
3. Run: `java Notepad`
